# LeetCode 1925. Count Square Sum Triples

## 1. 核心數學概念：枚舉與畢氏三元數 (Enumeration & Pythagorean Triples)

題目要求找出滿足 $1 \le a, b, c \le n$ 且 $a^2 + b^2 = c^2$ 的三元組 $(a, b, c)$ 的數量。
這就是尋找範圍內的**畢氏三元數 (Pythagorean triples)**。

由於 $a$ 和 $b$ 的順序不同視為不同的三元組（例如 $(3, 4, 5)$ 和 $(4, 3, 5)$ 是不同的），我們需要考慮所有可能的排列。

## 2. 解題策略：暴力枚舉 (Brute Force)

由於 $n$ 的範圍較小 ($1 \le n \le 250$)，我們可以使用 $O(n^2)$ 的暴力法來解決。

### 演算法步驟

1.  **雙重迴圈枚舉**：使用兩個迴圈分別枚舉 $a$ (從 1 到 $n$) 和 $b$ (從 1 到 $n$)。
2.  **計算 $c$**：計算 $c = \sqrt{a^2 + b^2}$。
3.  **檢查條件**：
    - 檢查 $c$ 是否為整數（即 $a^2 + b^2$ 是否為完全平方數）。
    - 檢查 $c$ 是否在範圍內 ($c \le n$)。
4.  **計數**：如果上述條件滿足，則計數器加一。
5.  **優化 (剪枝)**：在內層迴圈中，如果計算出的 $c$ 已經大於 $n$，則更大的 $b$ 也會導致更大的 $c$，因此可以提前結束內層迴圈 (`break`)。

## 3. 複雜度分析
- **時間複雜度**：$O(n^2)$，我們有兩個嵌套的迴圈，每個迴圈最多執行 $n$ 次。
- **空間複雜度**：$O(1)$，只需要常數空間來儲存變數。
