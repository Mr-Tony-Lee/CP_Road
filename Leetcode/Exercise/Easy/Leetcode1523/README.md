# LeetCode 1523. Count Odd Numbers in an Interval Range

## 1. 核心數學概念：奇偶性 (Parity)

我們的目標是計算在區間 $[low, high]$ 之間有多少個奇數。

我們可以觀察區間長度與奇數個數的關係：
- 區間內的數字總數為 $high - low + 1$。
- 如果區間內有偶數個數字，則奇數與偶數的數量相等。
- 如果區間內有奇數個數字，則奇數的數量取決於邊界的奇偶性。

## 2. 解題策略：數學推導

我們可以根據邊界條件歸納出公式：

1.  **若 `low` 或 `high` 其中之一為奇數**：
    - 只要邊界有一個是奇數，該區間包含的奇數個數就會比偶數多（或相等但進位）。
    - 例如 `low=3, high=7` (3, 4, 5, 6, 7)，奇數有 3, 5, 7 (共 3 個)。計算：$(7-3)/2 + 1 = 3$。
    - 例如 `low=3, high=6` (3, 4, 5, 6)，奇數有 3, 5 (共 2 個)。計算：$(6-3)/2 + 1 = 2$。
    - 此時奇數個數為 `(high - low) / 2 + 1`。

2.  **若 `low` 和 `high` 皆為偶數**：
    - 此時區間內的奇數個數會比偶數少一個（或剛好一半）。
    - 例如 `low=4, high=8` (4, 5, 6, 7, 8)，奇數有 5, 7 (共 2 個)。計算：$(8-4)/2 = 2$。
    - 此時奇數個數為 `(high - low) / 2`。

## 3. 複雜度分析
- **時間複雜度**：$O(1)$，直接使用數學公式計算。
- **空間複雜度**：$O(1)$，不需額外空間。
