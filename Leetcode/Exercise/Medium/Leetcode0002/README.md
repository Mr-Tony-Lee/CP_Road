# LeetCode 2. Add Two Numbers

## 1. 核心思維：模擬加法與直式運算

這道題目要求我們將兩個以反向鏈結串列表示的非負整數相加，並同樣以鏈結串列的形式回傳結果。
因為鏈結串列的開頭正好表示個位數，我們可以像在做「直式加法」一樣，順著鏈結串列依序由低位數加到高位數，並隨時記錄與處理**進位 (carry)** 。

## 2. 解題策略：虛擬頭節點 (Dummy Node) + 逐位數相加

### 為什麼使用虛擬頭節點 (Dummy Node)？

在建立新的鏈結串列時，我們常會需要處理「頭節點是否已建立」的邊界情況。使用一個預先宣告好的虛擬頭節點 `head = new ListNode(0)` 作為佔位符，可以讓我們在迴圈內無論何時，都能用統一的寫法 `now->next = new ListNode(...)` 來串接新節點，最後只要回傳 `head->next` 即可避開預設的第一個零節點，大幅簡化程式碼。

### 迴圈終止條件與進位處理

在走訪兩個鏈結串列 `l1` 與 `l2` 時，很有可能遇到兩者長度不同的情況。為了避免對空節點取值造成錯誤，我們需要：
1. **條件判斷防護**：只有當 `l1` 或 `l2` 不為空 (`nullptr`) 時，才把該節點的值加進總和 `sum`，並將指標向後移動。
2. **迴圈結束條件包容進位**：迴圈的繼續條件除了 `l1` 或 `l2` 還有節點未處理外，還要考慮**最後是否還有進位** (`carry > 0`)。例如 `9 + 1 = 10`，雖然兩個數字都走訪完了，但仍需要再多建一個節點來裝進位的 `1`。

## 3. 演算法步驟

1. **初始化**：
   - 建立一個虛擬頭節點 `head = new ListNode(0)`。
   - 建立一個指標 `now` 負責在串接新的節點，初始指向 `head`。
   - 宣告整數 `carry = 0` 紀錄每次相加產生的進位。
2. **走訪鏈結串列**：
   - 使用 `while` 迴圈，只要 `l1 != nullptr` 或 `l2 != nullptr` 或 `carry > 0` 至少一個條件成立就繼續執行。
   - 每次進入迴圈，先將當前的和 `sum` 設為上一輪留下來的 `carry`。
   - 若 `l1` 存在，將 `l1->val` 加入 `sum` 中，並將 `l1` 指標順勢移向 `l1->next`。
   - 若 `l2` 存在，將 `l2->val` 加入 `sum` 中，並將 `l2` 指標順勢移向 `l2->next`。
   - 算出新的 `carry = sum / 10`（供給下一位數加），以及此節點真正要存的值 `sum % 10`。
   - 建立包含新值 `sum % 10` 的節點，將它串接於 `now->next`，並讓 `now` 往後移動一位。
3. **回傳結果**：
   - 透過 `head = head->next` 跳過最初設定為佔位符的 `0` 節點。
   - 回傳 `head`，即為相加後的鏈結串列起點。

### 複雜度分析

- **時間複雜度**：$O(\max(M, N))$。其中 $M$ 和 $N$ 分別為兩個鏈結串列的長度。迴圈最多只會走訪較長的鏈結串列的長度，如果有最後的進位則跑 $\max(M, N) + 1$ 次，整理後為線性時間。
- **空間複雜度**：$O(\max(M, N))$。我們會動態建立一個新的鏈結串列來存放答案，其節點數量（長度）最多為 $\max(M, N) + 1$。
