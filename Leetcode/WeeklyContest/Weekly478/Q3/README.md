# Leetcode 3761: Minimum Absolute Distance Between Mirror Pairs

## 題目要求
給定一個整數陣列 `nums`，尋找一對索引 `(i, j)` 滿足 `j > i` 且 `nums[j]` 是 `nums[i]` 的「鏡像數」(Mirror)，並使得 `j - i` 最小。如果不存在這樣的對，則回傳 -1。
*   **鏡像數**：將一個數字的各個位數反轉後得到的數字 (例如 123 的鏡像數是 321)。

## 解法解釋
1.  **反轉函數**：實作 `reverse` 函數，將整數反轉。
2.  **記錄位置**：使用 `unordered_map<int, vector<int>>` 記錄每個數字在陣列中出現的所有索引位置。
3.  **遍歷尋找**：
    - 遍歷陣列中的每個元素 `nums[i]` (索引為 `i`)。
    - 計算其鏡像數 `r`。
    - 在 map 中查找 `r` 是否存在。
    - 如果存在，在 `r` 的索引列表 (`vector<int>`) 中使用 `upper_bound` 尋找第一個大於 `i` 的索引 `j`。
    - 如果找到這樣的 `j`，計算距離 `d = j - i`，並更新最小距離 `ans`。
4.  **回傳結果**：如果 `ans` 仍為初始最大值，表示沒找到，回傳 -1；否則回傳 `ans`。
