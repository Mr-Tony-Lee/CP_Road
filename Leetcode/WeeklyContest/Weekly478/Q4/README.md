# Leetcode 3762: Minimum Operations to Equalize Subarrays

## 題目要求
將子陣列中的元素變為相等或滿足特定條件所需的最小操作數。

## 解法解釋
本題使用了兩種進階的線段樹 (Segment Tree) 結構：

1.  **ModSegTree (區間極值線段樹)**：
    - 維護區間內的最小值 (`mn`) 和最大值 (`mx`)。
    - 支援區間查詢 `query(l, r)`，回傳該區間的最小值和最大值。

2.  **MergeSortTree (歸併排序樹)**：
    - 每個節點儲存該區間內所有元素的排序後列表 (`vals`) 以及前綴和 (`pref`)。
    - `build` 函數中，利用 `std::merge` 將左右子節點的有序列表合併。
    - **查詢功能 `queryLE(l, r, x)`**：
        - 計算在區間 `[l, r]` 中，小於等於 `x` 的元素個數以及這些元素的總和。
        - 利用 `upper_bound` 在節點的有序列表中快速定位，並結合前綴和計算總和。

這兩種資料結構通常用於處理複雜的區間統計問題，例如計算區間內元素與某個目標值的距離總和等。
*   ModSegTree：用於維護區間最小值與最大值。
*   MergeSortTree：用於處理區間內「小於等於某數」的個數與總和查詢。
